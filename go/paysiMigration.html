changes

  • authHeader changes
    grabapi auth header key is "x-mts-ssid" while paysi auth header key is "Authorization"
    so use the grabapi auth header key when calling grabapi

  • authHeader auth format changes
    paysi auth format is "GE accessToken:1", GrabAPI auth format can be "accessToken"(it can also be JWT token)
    so use GrabAPI auth format when calling grabapi
